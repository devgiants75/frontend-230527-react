1. TodoTemplate
  : 투두리스트의 레이아웃을 설정하는 컴포넌트 
  : 페이지 중앙에 그림자가 적용된 흰색 박스

2. TodoHead
  : 오늘의 날짜와 요일
  : 앞으로 해야 할 일이 몇개 남았는지 보여줌.

3. TodoList
  : 할 일에 대한 정보가 들어있는 todos 배열을 내장함수 map을 사용하여 여러개의 TodoItem 컴포넌트를 렌더링

4. TodoItem 
  : 각 할 일에 대한 정보를 렌더링해주는 컴포넌트
  : 할 일이 완료 > 좌측에 체크가 나타나고 텍스트의 색상이 연해짐.
  : 마우스를 올리면 휴지통 아이콘이 나타나고 이를 누르면 항목이 삭제

5. TodoCreate
  : 사용자가 새로운 할 일을 추가할 수 있는 기능을 제공
  : 버튼 클릭 시 입력 폼을 제공
  : 사용자가 정보를 입력한 뒤에 제출하면, 이 정보가 .TodoList에 반영되어야 함.
